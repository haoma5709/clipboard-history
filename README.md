# 历史剪贴板

一个轻量的 Windows 剪贴板工具。记录你复制过的文字和图片，随时调用历史重新粘贴。

## 安装

找到releases，点开历史剪贴板，下载1.0.0.exe，报告不安全时，选择保留即可下载。
双击exe即可运行，无需安装。
启动后图标出现在系统托盘（右下角），开机自动启动。

## 使用

| 操作 | 方式 |
|------|------|
| 查看历史 | 按 `Ctrl+Alt+V` 或点击托盘图标 |
| 粘贴某条记录 | 点击对应的记录，按ctrl+v即可粘贴 |
| 隐藏窗口 | 点击窗口外任意位置 或 按 `Esc` |
| 退出程序 | 右键系统托盘图标 → 退出 |

## 特性

- 自动记录文字和图片
- 保留最近 3 天、最多 100 条
- 窗口在鼠标旁弹出，失焦自动隐藏
- 完全本地运行，数据不上传
